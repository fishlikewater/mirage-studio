# Trellis 规范索引

本目录承接原 `AGENTS.md` 中的项目技术规范。根目录 `AGENTS.md` 只保留协作约定、项目定制位和流程入口；稳定的架构、实现、验证、发布与扩展规则以本目录为准。

## 推荐阅读顺序

1. [项目总览](./project-overview.md)
2. [前端规范](./frontend/index.md)
3. [后端规范](./backend/index.md)
4. [思考指引](./guides/index.md)

## 维护原则

- 技术事实和可执行规则放在 `.trellis/spec/`。
- 行为变更规格放在 `openspec/`。
- 设计稿和实现计划放在 `docs/superpowers/`。
- 根 `AGENTS.md` 不记录易变模块清单、详细扩展步骤或验证细节。
